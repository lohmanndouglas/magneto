Ajuda

  • Utilizar o mouse para movimentar-se na tela e selecionar objetos.

  •    +     para inserir objetos e cargas de prova.

  •   {  }   para calcular as variáveis E(campo eletrico), F(força), V(potencial) e W(trabalho).

  • Duplo clique sobre o objeto para alterar as configurações.
