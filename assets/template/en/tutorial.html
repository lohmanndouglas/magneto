Hi,


With this application you will be able to simulate problems involving Magnetism.


Magneto is a totally free open-source project, so you can also contribute with this project by github.


The first step with this simulator is to familiarize yourself with the interface. So I recommend you read this brief introduction and if you still have questions, you can consult the help menu, or access the project website.


You can also change the language of the simulator by accessing the settings, in this menu it is also possible to insert Cartesian planes (xOy, yOz and xOz) for notion of measurements in the three-dimensional plane and the axes (x, y and z). In addition, the accuracy of the calculations can be adjusted up to 6 decimal places.

First Simulation

 

Using this application, you can obtain the Magnetic Field at a defined point in space and the Magnetic Force that a charge is submitted. Before calculating some of these variables you must add one or more objects (lines or turns of current) and one or more charges (which will indicate the points in which you want to get the value of the variable Field or Force). To add these objects click the + menu and then add objects and a load, defining the values ​​of electric current, position (x, y, z), rotation in the axes (θx, θy, θz), velocity (vx, vy, vz ) and the charge value (velocity and charge values ​​will be used only to calculate the magnetic force).


Having defined the objects and the charge you can perform the calculation of the available variables by clicking on the { } menu. Note that after performing the calculation a box will be displayed next to the charge as well a vector that determines the module, direction and direction of the calculated variable. If you click on the box you can see the value of the components of the calculated vector.


Now you can start the simulations of Physics on Field and Magnetic Force.


Have fun!
